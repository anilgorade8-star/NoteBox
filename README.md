# 📝 Notely – React Notes App

Notely is a simple and elegant notes application built using **React.js**.
It allows users to quickly add, view, and delete notes with a clean and user-friendly interface inspired by handwritten sticky notes.

---

## 🚀 Features

* ➕ Add new notes
* 🗑️ Delete notes easily
* 🎨 Clean and minimal UI
* 📱 Responsive design
* ⚡ Fast and lightweight (React-based)

---

## 🛠️ Tech Stack

* **React.js**
* **HTML5**
* **CSS3**
* **JavaScript (ES6+)**


## 💡 Future Improvements

* ✏️ Edit notes feature
* 🌙 Dark mode
* 💾 Save notes using LocalStorage
* 🔍 Search and filter notes
